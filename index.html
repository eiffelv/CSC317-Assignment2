How to Apply CSS to HTML

External CSS may be applied to an HTML file using the <link> tag. The <link> tag is placed in the <head> section of the HTML file. The <link> tag needs two attributes for applying External CSS: rel and href. For example:

<link rel="stylesheet" href="style.css">

How to Use JavaScript in a Webpage

JavaScript may be applied to an HTML file using the <script> tag. The <script> tag may be placed in the <head> or <body> section of the HTML file. The <script> tag may contain JavaScript code or link to an external JavaScript file.

To link to an external JavaScript file, use the src attribute. For example:

<script src="script.js"></script>
To write JavaScript code directly in the HTML file, use the <script> tag with the code inside. For example:
<script> alert("Hello, World!"); </script>
